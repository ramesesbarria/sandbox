# PythonAA Module Deployment

## Overview
`PythonAA` is a reusable Python module for Automation Anywhere that:
- Saves Word documents
- Logs actions to `logs.txt`
- Returns JSON string output for AA
- Bootloads paths and debug mode from `.env`

## Setup Instructions

1. **Install Python**  
   Python 3.10+ recommended.
